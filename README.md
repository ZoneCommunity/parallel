### How to use parallel
1. **Clone the repository**
   - ```git clone https://github.com/ZoneCommunity/parallel```
2. **Navigate into the root of the repository**
   - ```cd parallel```
3. **Install dependencies and run parallel**
   - ```npm install && npm start```
